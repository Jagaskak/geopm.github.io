Detailed Description

Simple implementation of a power balancing tree decider.

The balancing decider uses the runtimes of each child node to calculate ratios of power to give to each node. Nodes that are slower will be given more power than nodes that are ahead. The sum of the individual node budgets will sum to the budget allocated to the level of the hierarchy the decider instance is running at.
