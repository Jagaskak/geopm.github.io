virtual std::vector<int> geopm::IProfileSampler::cpu_rank ( void  ) const
pure virtual

Retrieve a vector to the affinities of all application ranks.

Return vector is sized to number of Linux online CPUs in the system. Each element of the vector is indexed by the Linux CPU ID, and the value assigned is the MPI rank running on the CPU (or -1 if no rank has been affinitized).

Returns
Vector to be filled with the MPI rank for each Linux CPU, set to -1 if no MPI rank is affinitized.

Returns the samples present in all the per-rank hash tables.

Fills in a portion of a vector which is assumed to be already sized greater than or equal to the maximum number of samples we can return. This value can be queried with the capacity() method.

Parameters
[out]contentVector to be filled with per-node sample messages.
[out]lengthThe number of samples that were inserted.
[in]commComm object required for barriers in handshake with application.
