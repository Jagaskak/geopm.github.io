Detailed Description

Encapsulates the power policy that is applied across the entire MPI job allocation being managed by geopm.

The C++ implementation of the resource manager interface to the GEOPM policy. The class methods support the C interface defined for use with the geopm_policy_c structure and are named accordingly. The geopm_policy_c structure is an opaque reference to the geopm::GlobalPolicy class.
