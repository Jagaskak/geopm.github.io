void geopm_error_destroy_shmem ( void  )

Unlinks all shared memory keys in case of error.

Looks in /dev/shm for any keys starting with "/geopm-shm" or the GEOPM_SHMKEY environment variable if set. All matching keys are unlinked with shm_unlink().
