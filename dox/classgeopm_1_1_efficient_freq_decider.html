Detailed Description

Energy efficient implementation of a binary frequency decider.

This frequency decider uses the geopm_hint interface or feedback from region runtime obtained offline or online to determine whether we are in a compute or memory bound region and choose the maximum frequency and a fraction of the minimal possible frequency repsectively.

This is a leaf decider.
