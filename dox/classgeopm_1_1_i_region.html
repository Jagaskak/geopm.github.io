Retrieve the derivative of the signal values for a domain of control.

Get the derivative of the signal values for a given domain of control and a signal type for the buffered data associated with the application region. There must be at least 2 valid samples recorded.

Parameters
[in]domain_idxThe index to the domain of control as ordered in the Platform and the Policy.
[in]signal_typeThe signal type requested as enumerated in geopm_signal_type_e in geopm_message.h.
Returns
If there are 2 valid samples then return he derivative of the signal values, else return NAN.
