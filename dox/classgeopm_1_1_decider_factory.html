IDecider * geopm::DeciderFactory::decider ( const std::string &  description)
virtual

Returns an abstract Decider pointer to a concrete decider.

The concrete Decider is specific to the description string passed in to select it. throws a std::invalid_argument if no acceptable Decider is found.

Parameters
[in]descriptionThe description string corresponding to the desired Decider.
